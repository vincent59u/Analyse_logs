Installation


Eléments requis

  1. JDK Java 7+
  2. Apache ANT

Installation à partir de la distribution source

  1. Télécharger la distribution source (<projet><version>-src.tar.gz)
  2. Décompresser le tout dans une dossier de votre choix (exemple : c:\)
  3. Aller dans le dossier <projet> et lancer la commande ant
    4. > ant

Désinstallation

Supprimer le dossier d'installation
